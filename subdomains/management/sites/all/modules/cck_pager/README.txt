DESCRIPTION:
------------
Allows cck fields with multiple values to be paginated.


INSTALLATION:
-------------
1. Place the entire cck_pager directory into your Drupal sites/all/modules/
   directory.

2. Enable the cck_pager module by navigating to:

     administer > modules


Features:
---------
CCK Pager allows to page the item list of any field with multiple values.
The module reimplements all the formatters defineds by all the others field
and extends them by paging the field item list; setting the number of elements 
per page can be made globally or per field.
